Note di installazione

Alenface è attualmente nella sua fase di bootstrap (non firmata). Per eseguire l'app scaricata, rimuovi il blocco di macOS Gatekeeper tramite Terminale:

xattr -cr /Applications/Alenface.app

Avvio in una riga tramite Terminale:

xattr -c /Applications/Alenface.app && open -a Alenface
