Phase Zero Installation

Alenface is currently in a bootstrap phase. To run this unsigned test build, follow these steps:

# 1. Install to your Applications folder
# 2. Open Terminal and clear macOS Gatekeeper:
xattr -cr /Applications/Alenface.app

Learn more about xattr: ss64.com/mac/xattr.html [cite: 47]
