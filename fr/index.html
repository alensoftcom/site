Note d'installation

Alenface est actuellement dans sa phase de démarrage (non signé). Pour exécuter l'application téléchargée, nettoyez Gatekeeper de macOS via le terminal :

xattr -cr /Applications/Alenface.app

Démarrage en une ligne via le terminal :

xattr -c /Applications/Alenface.app && open -a Alenface
