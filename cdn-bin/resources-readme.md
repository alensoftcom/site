# Алгоритм

1. В файле resources.hash фиксируются хеши файлов.
2. При изменении хеша тело шифруется заново
3. Само шифрование каждый раз УНИКАЛЬНО и поэтому нельзя шифровать повторно
4. Поэтому нельзя !!! удалять resources.hash - при удалении будет перешифровка
5. В приложении ресурсы лежат без шифрования
