Заметка по установке

Alenface сейчас находится на стадии Bootstrap (без подписи). Чтобы запустить скачанное приложение, очистите macOS Gatekeeper через Терминал:

xattr -cr /Applications/Alenface.app

Запуск одной строкой через Терминал:

xattr -c /Applications/Alenface.app && open -a Alenface
