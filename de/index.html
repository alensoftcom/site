Installationshinweis

Alenface befindet sich derzeit in der Bootstrap-Phase (unsigniert). Um die heruntergeladene App auszuführen, entfernen Sie das macOS Gatekeeper-Attribut über das Terminal:

xattr -cr /Applications/Alenface.app

Start in einer Zeile über das Terminal:

xattr -c /Applications/Alenface.app && open -a Alenface
